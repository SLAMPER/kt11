Задание 1

Задание: Напишите обобщенный метод Swap<T>(ref T x, ref T y), который меняет местами значения двух переменных типа T. Затем напишите ограничение для этого метода, чтобы он мог работать только с типами, которые являются типами значений (struct). Затем напишите пример вызова этого метода для переменных типов int, double и bool.
