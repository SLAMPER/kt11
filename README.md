Задание 3

Задание: Напишите обобщенный интерфейс IPrintable<T>, который содержит метод void Print(), который выводит на консоль информацию об объекте типа T. Затем напишите ограничение для этого интерфейса, чтобы он мог работать только с типами, которые являются ссылочными типами (class) или типами значений (struct), но не с указателями (pointer) или динамическими типами (dynamic). Затем напишите классы Student и Vector, которые реализуют этот интерфейс и выводят на консоль свои свойства, такие как Name, Age, Grade, X, Y и Z. Затем напишите пример использования этого интерфейса для печати объектов этих классов.
